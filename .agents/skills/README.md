# Antigravity Skills

This folder is used to define custom skills for Antigravity as described in [docs/skills](https://antigravity.google/docs/skills).

### How to use

Each skill should be a folder containing a `SKILL.md` file. For example:

```
.agents/skills/my_skill/
├── SKILL.md
└── scripts/
    └── my_script.js
```

The AI can then refer to these skills to enhance its capabilities.
